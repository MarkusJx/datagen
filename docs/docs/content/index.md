title: Generators

Datagen comes with a set of generators that can be used to generate data.
You can also create your own generators.

## Available generators

* [`array`](array.md)
* [`object`](object.md)
* [`integer`](integer.md)
* [`number`](number.md)
* [`string`](string.md)
* [`bool`](bool.md)
* [`anyOf`](anyof.md)
* [`flatten`](flatten.md)
* [`plugin`](plugin.md)
* [`reference`](reference.md)

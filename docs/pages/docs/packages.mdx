# Packages

`datagen` consists of multiple packages:

- Node.js packages:
  - [`@datagen-rs/cli`](packages/nodeCli.md) - The node.js CLI for `datagen`
  - [`@datagen-rs/node`](packages/node.md) - Node.js bindings for `datagen`
  - [`@datagen-rs/types`](packages/nodeTypes.md) - TypeScript types for `datagen`
